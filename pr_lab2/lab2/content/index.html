How to Use

  1. Open one of the sample files (e.g., /image.png) in a new tab.
  2. Click Run to fire parallel requests to the same path.
  3. Watch counters (top-left) update and compare timings with/without locks.
Try two server variants:
  • No Lock: expect occasional inconsistent counters.
  • With Lock: counters remain consistent under load.
